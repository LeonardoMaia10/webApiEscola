## 📚 Como baixar o projeto

```bash

# Clonar o repositório
$ git clone https://github.com/LeonardoMaia10/webApiEscola

# entrar no diretório
$ cd escola

#instalar as dependencias
$ dotnet install

# iniciar o projeto
$ dotnet start
```
